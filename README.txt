This is plugin to create simple contact form.

== Description ==
With CS Contact Form, you can create an manage multiple contact form. This plugin support customize form contact very easily. Moreover, the form support send mail, check validate, reCAPTCHA. The form is styled basic to display on Front-end. 

### Features of free version

* Easy install and use
* Allows you to easily create WordPress contact forms in just a few minutes without writing any code.
* Works perfectly with all versions of WordPress
* Fully Responsive
* Add Unlimited fields
* Required option for any field you need
* Easily move fields positions
* Option apply Google reCaptcha for your form
* Config Recipient email and receive mail from contact form.


### Features of Pro version

* All features of free version
* There is a design template for contact page
* Ajax validate for form contact
* Allow you to save Client contact content to database and display in dashboard. If you don't like that feature, you can turn off it
* You can add more information (Address, Phone, Email, Website) to display on contact page
* You can add Google Map and display on contact page.
* You can add and display social channel (YouTube, Facebook, Twitter, Instagram, Linkedin, Pinterest).
* You can add and display chat message (WhatsApp, Skyper)

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "CS Contact Form"
3. **Install and Activate** that plugin in your Plugins page
4. You will find **CS Contact** menu in your WordPress admin screen.
5. **Read** the documentation to [get started] http://codesamples.info/guide
